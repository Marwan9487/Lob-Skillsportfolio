# Mijn Skillsportfolio LOB

Welkom op mijn loopbaan en vaardighedenportfolio. Gedurende mijn ICT-opleiding gebruik ik deze repository om mijn professionele groei, studievaardigheden en loopbaancompetenties bij te houden en te bewijzen.

Over Mij
Naam: Marwan
Opleiding: ICT Software Development

Je visie op de ICT-professional: Een goede ICT'er beschikt niet alleen over technische skills, maar heeft ook een professionele beroepshouding: proactief, communicatief vaardig en eigenaar van zijn of haar eigen leerproces.

Mijn Leerdoelen en Status

Hieronder houd ik bij hoe ik scoor op de hoofdleerdoelen van deze periode:

Leerdoel: Verantwoordelijkheid voor eigen leerproces
Status: Ik plan mijn eigen taken in, maar vind hulp vragen soms nog lastig.

Leerdoel: Persoonlijke kwaliteiten benoemen
Status: Ik weet dat ik analytisch ben, maar moet leren dit met voorbeelden uit te leggen.

Leerdoel: Studievaardigheden inzetten
Status: Ik gebruik de Pomodoro-techniek voor focus en plan via Trello.

Leerdoel: Reflecteren op verschillende manieren
Status: Ik ga deze periode experimenteren met de STARR-methode.

Leerdoel: Beroepshouding en Beroepsbeeld
Status: Ik onderzoek wat er van een junior developer wordt verwacht in het bedrijfsleven.

De 5 Loopbaancompetenties

Hieronder vind je de snelkoppelingen naar mijn uitwerkingen per competentie:

1. Kwaliteitenreflectie Wat kan ik goed?
2. Motievenreflectie Waar ga en sta ik voor?
3. Werkexploratie Waar ben ik het beste op mijn plek?
4. Loopbaansturing Hoe bereik ik mijn doel?
5. Netwerken Wie kan mij helpen mijn doel te bereiken?
